🔧 주요 기능

📝 상품 등록

상품명, 가격, 설명, 카테고리 등 상세 정보를 입력하여 상품을 등록할 수 있습니다.

✅ 입력값 검증

필수값 검증, 형식 검증, 데이터 타입 검증을 통해 올바른 데이터만 입력되도록 합니다.

🎯 커스텀 검증

비즈니스 로직에 맞는 커스텀 검증 규칙을 적용하여 데이터 품질을 보장합니다.

🌐 다국어 지원

에러 메시지와 시스템 메시지가 국제화를 지원하여 다양한 언어로 제공됩니다.

🧪 테스트 가이드

기본 검증 테스트

  • 모든 필드 비우기 → 에러 메시지 확인
  • 가격에 0 또는 음수 입력
  • 필수 필드 누락 시 동작 확인

커스텀 검증 테스트

  • 상품명에 "금지어" 포함하기
  • 전자제품 카테고리 + 10,000원 미만 가격
  • 비즈니스 로직 검증 확인

성공 시나리오

  • 모든 필드 올바르게 입력
  • 성공 페이지로 이동 확인
  • 한글 처리 정상 동작 확인

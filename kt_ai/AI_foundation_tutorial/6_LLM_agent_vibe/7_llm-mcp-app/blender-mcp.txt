# MCP (Motion Capture Panel)

## 개요
MCP (Motion Capture Panel)는 Blender에서 모션 캡처 데이터를 처리하고 적용하기 위한 애드온입니다. 이 도구는 주로 BVH 파일 형식의 모션 캡처 데이터를 Blender의 아마추어 리깅 시스템에 적용하는 데 사용됩니다.

## 주요 기능
1. **모션 캡처 데이터 불러오기**: BVH 파일 형식의 모션 캡처 데이터를 Blender로 불러올 수 있습니다.
2. **리깅 시스템과 매핑**: 모션 캡처 데이터의 본(bone)을 Blender의 아마추어 리깅 시스템의 본과 매핑할 수 있습니다.
3. **애니메이션 편집**: 불러온 모션 캡처 데이터를 편집하고 수정할 수 있습니다.
4. **T-포즈 조정**: 모션 캡처 데이터의 기본 포즈를 T-포즈로 조정할 수 있습니다.
5. **리타겟팅(Retargeting)**: 다른 골격 구조를 가진 캐릭터에 모션 캡처 데이터를 적용할 수 있습니다.

## 작동 방식
1. **설치**: Blender의 Add-ons 설정에서 MCP 애드온을 설치합니다.
2. **패널 접근**: 설치 후 3D 뷰포트의 사이드바(N키)에서 MCP 패널에 접근할 수 있습니다.

3. **모션 캡처 데이터 불러오기**:
   - BVH 파일을 선택하여 불러옵니다.
   - 불러온 모션 캡처 데이터는 Blender에서 새로운 아마추어 객체로 생성됩니다.

4. **리타겟팅 과정**:
   - 소스 아마추어(모션 캡처 데이터)와 타겟 아마추어(캐릭터 리그)를 선택합니다.
   - 본 매핑을 설정합니다. 이는 소스 아마추어의 본과 타겟 아마추어의 본 사이의 관계를 정의합니다.
   - FK(Forward Kinematics) 또는 IK(Inverse Kinematics) 방식을 선택합니다.
   - 리타겟팅 프로세스를 실행하여 모션 캡처 데이터를 타겟 아마추어에 적용합니다.

5. **애니메이션 조정**:
   - 프레임 범위, 스케일, 오프셋 등을 조정하여 애니메이션을 수정할 수 있습니다.
   - 특정 본의 움직임을 개별적으로 조정할 수도 있습니다.

## 활용 사례
- **게임 개발**: 게임 캐릭터의 자연스러운 움직임을 구현합니다.
- **애니메이션 영화 제작**: 캐릭터 애니메이션의 기초로 모션 캡처 데이터를 사용합니다.
- **VR/AR 콘텐츠**: 가상 현실이나 증강 현실에서 캐릭터의 움직임을 구현합니다.

## 고급 기능
- **본 제약 조건(Constraints)**: 특정 본의 움직임에 제약 조건을 적용하여 더 자연스러운 동작을 구현할 수 있습니다.
- **액션 결합(Action Blending)**: 여러 모션 캡처 데이터를 혼합하여 새로운 애니메이션을 만들 수 있습니다.
- **애니메이션 레이어(Animation Layers)**: 여러 레이어의 애니메이션을 관리하고 적용할 수 있습니다.

## 주의사항
- 모션 캡처 데이터와 타겟 리그의 본 구조가 크게 다를 경우, 리타겟팅 과정에서 문제가 발생할 수 있습니다.
- 높은 품질의 결과를 위해서는 본 매핑을 정확하게 설정하는 것이 중요합니다.
- 복잡한 동작이나 물리적 상호작용을 포함하는 모션 캡처 데이터는 추가적인 수작업이 필요할 수 있습니다.
